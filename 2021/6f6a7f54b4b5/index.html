zfill函数

功能

为字符串定义长度, 如不满足,缺少的部分用0填充

用法

newstr = string.zfill(width)

参数

width: 新字符串希望的宽度

注意事项

  • 与字符串的字符无关
  • 如果定义长度小于当前字符串长度,则不发生变化

字符串的count函数

功能

返回当前字符串中某个成员(元素)的个数

用法

inttpe = string.count(item)

参数

item: 查询个数的元素

注意事项

  • 返回的是整形
  • 如果查询的成员(元素)不存在,则返回0
