### 编写程序

创建并打开`HelloWorld.java`并书写以下内容

```java
public class HelloWorld {
	public static void main(String[] args) {
		System.out.println("Hello World");
	}
}
```

#### 编译并运行程序

1. **进入文件根目录**
2.  `javac HelloWorld.java`
3. `java HelloWorld`

#### 输出结果

```
Hello World
```
