列表(元组)的count函数

功能

  • 返回当前列表中某个成员的个数

用法

  • inttype = list.count(item)

参数

  • item : 你想查询个数的元素

注意事项

  • 如果查询的成员(元素)不存在,则返回 0

  • 列表只会检查完整元素是否存在需要计算的内容

  • 列表和元组的count是一致的
