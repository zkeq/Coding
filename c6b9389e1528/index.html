整形

int 定义整形 ,又是定义整形的 内置函数

定义一个整型,并不一定要使用int

cout_100_01 = int(100)

cout_100_02 = 100

不要使用long

浮点型

凡是带有小数点的类型,都可以认为是浮点型

pi_01 = float(3.14)

pi_02 = 3.14

定义一个浮点型,并不一定要使用float

内置函数 type

字符串 类型

什么是字符串?

用 ‘ ‘ 或者 “” 包裹的信息 就是 字符串

字符串中可以包括任意字符且没有先后顺序

字符串定义方法

使用str定义

safe = str(‘健康的体温是36.5左右’)

字符串不可改变!!!

name = ‘dewei’
